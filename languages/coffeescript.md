シンタックスルール
==================

インデント
----------

- 半角スペース2つ分でインデントする。
- タブを使ったり、タブとスペースを混在させるのは NG。


行末のスペース
--------------

- 行末のスペースを削除する。


セミコロン
----------

- 文末のセミコロン `;` は除去する。


コンマ
------

- 省略できるものは、すべて省略する。

```coffee
# OK
foo = [
  'some'
  'string'
  'values'
]
bar:
  label: 'test'
  value: 87

# NG
foo = [
  'some',
  'string',
  'values'
]
bar:
  label: 'test',
  value: 87
```




スタイルルール
==============

コメント
--------

- 意味単位のブロックでタイトルコメントを記述する。
- タイトルコメントは以下のスニペットを使用すること。

```coffee
#=== 大見出し
#==============================================================================================
```

```coffee
#  少見出し
#-----------------------------------------------
```
